原理：通过将图像分成16×16的像素块，并将像素块的内容压缩进张量中送入原有的transformer结构，使得视觉问题变为seq2seq问题，使用特殊的1，2d位置编码，注意力之后是mlp的堆叠，用特殊字符对应的输出作为encoder输出，最后mlp图片分类。当然不一定需要特殊字符，可以在平均池化后直接获取对应效果，两种位置编码也进行了验证1d略有改善，1，2d相差不大。

实验：设计了vit的三种大小，resnet，resnet50输出的特征图再进行vit的对比试验，验证了不同大小数据集上的表现，随着模型的增大，vit变得越来越好



可视化分析：

先PCA分析了28个主成分，可以看到**底层特征与 CNN 高度趋同**：左上角的滤波器呈现出明显的边缘、纹理、频率检测模式（竖线、横线、棋盘格），和 CNN 底层卷积核学到的基础视觉特征几乎完全一致。

- 这证明：即使 ViT 没有硬编码 CNN 的「局部性、平移不变性」归纳偏置，也能通过大规模数据预训练，自动学习到视觉任务必需的基础特征提取能力。

**特征层级化规律**：从左上到右下，滤波器的模式从简单的边缘纹理，逐渐变为带颜色、带形状的复杂模式，和 CNN 的层级特征学习逻辑一致 —— 底层学基础视觉元素，高层学复杂语义特征。

**关键意义**：这张图直接回应了当时对 ViT 的核心质疑 ——「Transformer 没有卷积的归纳偏置，无法学好视觉特征」，用可视化结果证明了 Transformer 在视觉任务上的有效性。



Position embedding similarity：对每个patch的位置向量进行向量积计算余弦，发现距离越近越相似，即使是1d的编码也学到了2d的信息



层数-注意力距离分析

对单张输入图像，计算每个 query patch 与所有 key patch 的空间像素距离；

用该 query patch 对应的注意力权重，对所有 key patch 的距离做加权平均，得到该 query 的注意力距离；

对所有测试图像、所有 query patch、所有样本取平均，最终得到该注意力头在对应层的平均注意力距离。

我们观察到在浅层，每个头都在观察不同尺寸的信息，同时在深层，他们将他们的特征滤波到了全局，使之能够优化到不与全局冲突

*多头是将全部的输入糅合进64维而不是直接切片

*浅层到深层感受野扩大的原因是，多层transformer成功融合了其邻居信息，使得能够观测的范围越来越大，全局原生头则一直保持着信息统合，发挥着关键作用

*其重点意义在于其论证了假设偏置仅在小数据集上泛化能力较强，大数据集上，似乎越简单的结构越有优势，更容易从数据中学到通用规律

*我猜想，越高分辨率越好的原因，对于后续极其尖锐的局部最小值可以减少
